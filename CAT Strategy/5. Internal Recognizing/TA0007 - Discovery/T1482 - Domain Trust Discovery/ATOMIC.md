<h2><a aria-hidden="true" class="anchor" href="#description-from-attck" id="user-content-description-from-attck"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z" fill-rule="evenodd"></path></svg></a><a href="https://attack.mitre.org/wiki/Technique/T1482" rel="nofollow">Description from ATT&amp;CK</a></h2>
<blockquote>Adversaries may attempt to gather information on domain trust relationships that may be used to identify [Lateral Movement](<a href="https://attack.mitre.org/tactics/TA0008" rel="nofollow">https://attack.mitre.org/tactics/TA0008</a>) opportunities in Windows multi-domain/forest environments. Domain trusts provide a mechanism for a domain to allow access to resources based on the authentication procedures of another domain.(Citation: Microsoft Trusts) Domain trusts allow the users of the trusted domain to access resources in the trusting domain. The information discovered may help the adversary conduct [SID-History Injection](<a href="https://attack.mitre.org/techniques/T1178" rel="nofollow">https://attack.mitre.org/techniques/T1178</a>), [Pass the Ticket](<a href="https://attack.mitre.org/techniques/T1097" rel="nofollow">https://attack.mitre.org/techniques/T1097</a>), and [Kerberoasting](<a href="https://attack.mitre.org/techniques/T1208).(Citation" rel="nofollow">https://attack.mitre.org/techniques/T1208).(Citation</a>: AdSecurity Forging Trust Tickets)(Citation: Harmj0y Domain Trusts) Domain trusts can be enumerated using the DSEnumerateDomainTrusts() Win32 API call, .NET methods, and LDAP.(Citation: Harmj0y Domain Trusts) The Windows utility [Nltest](<a href="https://attack.mitre.org/software/S0359" rel="nofollow">https://attack.mitre.org/software/S0359</a>) is known to be used by adversaries to enumerate domain trusts.(Citation: Microsoft Operation Wilysupply)</blockquote>

<p>Uses the dsquery command to discover domain trusts.
Requires the installation of dsquery via Windows RSAT or the Windows Server AD DS role.</p>
<p><strong>Supported Platforms:</strong> Windows</p>
