<blockquote>Adversaries will likely look for details about the network configuration and settings of systems they access or through information discovery of remote systems. Several operating system administration utilities exist that can be used to gather this information. Examples include [Arp](<a href="https://attack.mitre.org/software/S0099" rel="nofollow">https://attack.mitre.org/software/S0099</a>), [ipconfig](<a href="https://attack.mitre.org/software/S0100)/[ifconfig](https://attack.mitre.org/software/S0101" rel="nofollow">https://attack.mitre.org/software/S0100)/[ifconfig](https://attack.mitre.org/software/S0101</a>), [nbtstat](<a href="https://attack.mitre.org/software/S0102" rel="nofollow">https://attack.mitre.org/software/S0102</a>), and [route](<a href="https://attack.mitre.org/software/S0103" rel="nofollow">https://attack.mitre.org/software/S0103</a>).</blockquote>
